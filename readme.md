## Features

## Requirements
- Node.js 14+ (or newer)
- npm

## Quick start
1. Change to the server directory:
    ```bash
    cd server
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the server:
    ```bash
    npm run dev
    ```
4. Open the client in a browser on any machine in the same network:
    - http://&lt;server-ip&gt;:3001
